What is Dockside?

Dockside lets dev teams code within production environments from their browser - using a powerful online IDE launched in development containers - then share and stage their work with stakeholders.

{% octicon home height:28 class:"fill-blue d-inline mr-2" %} Easily work from anywhere

Dockside is lightweight and accessible within a browser, making working from anywhere within a production environment possible.

{% octicon rocket height:28 class:"fill-blue d-inline mr-2" %} Fast and lean

Spend less time on setting up your development environment. Launch a devtainer for every task, fork or branch, with only a couple of clicks, within seconds.

{% octicon share height:28 class:"fill-blue d-inline mr-2" %} Instantly share work with stakeholders

Stage any devtainer instantly so its services can be shared securely over HTTPS, either privately or on the public internet, whether for review, sign-off or testing, by colleagues, clients or management.

{% octicon bug height:28 class:"fill-blue d-inline mr-2" %} Eliminate errors

With Dockside you do not have to worry about differences between development and production environments.

{% octicon lock height:28 class:"fill-blue d-inline mr-2" %} Safe and powerful

Provide power users with access to underlying system resources and alternative runtimes and architectures, while locking down access for other users through a role-based access control scheme.

{% octicon infinity height:28 class:"fill-blue d-inline mr-2"aria-label: %} Low footprint

Dockside’s resource requirements are minimal. It is proven to support large numbers of devtainers on relatively small servers, VMs or cloud instances.

{% octicon code height:28 class:"fill-blue d-inline mr-2" %} Powerful VS Code-compatible IDE

Ships with out-of-the-box support for the incredible open-source Eclipse Theia IDE, which is powerful and VS Code-compatible

{% octicon checklist height:28 class:"fill-blue d-inline mr-2"aria-label: %} Commercial-friendly open-source licence

Free and open-source under the Apache 2.0 licence: so suitable for use in bespoke commercial environments and can be self-hosted locally, on-prem or in the cloud.

Get started today

Try Dockside locally within minutes. All you need is Docker.

1. If you have Docker running locally, install Dockside by running:
mkdir -p ~/.dockside && \
docker run -it --name dockside \
-v ~/.dockside:/data \
-v /var/run/docker.sock:/var/run/docker.sock \
-p 443:443 -p 80:80 \
newsnowlabs/dockside --ssl-builtin
2. Navigate to https://www.local.dockside.dev/ in your browser.
  • 3. Sign in as admin with the auto-generated password that is output to the terminal.
4. Follow the on-screen instruction to launch your first devtainer!
